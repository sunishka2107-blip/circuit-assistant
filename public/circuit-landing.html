DROP-IN INTEGRATION

Add Nova to any site in two lines.

Host circuit-widget.html and circuit-embed.js wherever your other static assets live, then choose one of the two integration paths below.

Floating launcher (recommended)

Adds a mic bubble that opens Nova in a corner panel — like a chat widget.

<script src="https://YOUR-HOST/circuit-embed.js"
        data-src="https://YOUR-HOST/circuit-widget.html"
        data-position="bottom-right"
        data-accent="#e8a33d"></script>

Inline iframe

Embed Nova directly inside a page section, like the live demo above.

<iframe
  src="https://YOUR-HOST/circuit-widget.html"
  title="Nova electrical assistant"
  allow="microphone"
  style="width:100%;max-width:420px;height:640px;
         border:none;border-radius:14px;">
</iframe>
AttributeApplies toDescription
data-srccircuit-embed.jsRequired. Full URL where circuit-widget.html is hosted.
data-positioncircuit-embed.jsbottom-right (default) or bottom-left.
data-accentcircuit-embed.jsHex color for the launcher bubble, default #e8a33d.
allow="microphone"iframeRequired for voice input to work inside the frame.

Both paths load circuit-widget.html in its own frame, so it never inherits the host site's styles and can't be broken by the host page's CSS or scripts. Voice input requires the page to be served over https:// and the browser to grant microphone permission to the frame.
